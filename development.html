Development¶

We are happy about contributions. How To Contribute should provide all information you need.

Help keeping PyInstaller alive: Maintaining PyInstaller is a huge amount of work. PyInstaller development can only continue if users and companies provide sustainable funding. See Funding PyInstaller for how to support PyInstaller.

Source Code Repository¶

  • You may browse the current repostory at the Repository Browser.

  • Or you may check out the current version by running:

    git clone https://github.com/pyinstaller/pyinstaller.git

If you are interested in the “core” development, including tools and processes, please refer to the Development Guide in the manual.

Maintainers¶

PyInstaller is currently maintained by Hartmut Goebel.

Help keeping PyInstaller alive: Maintaining PyInstaller is a huge amount of work. PyInstaller development can only continue if users and companies provide sustainable funding. See Funding PyInstaller for how to support PyInstaller.

The project was founded by Giovanni Bajo (rasky@develer.com, @giovannibajo). Former (co-)maintainers and major contributors include Martin Zibricky, David Cortesi, and David Vierra.

We would like to thank Gordon McMillan who wrote the original Python Installer, and William Caban for his initial development and maintenance effort without which PyInstaller would not exist today.
